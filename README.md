# Egg Wars 3D

**A 3D browser-based educational game where you solve math and spelling challenges to defend your egg!**

Created by **J_ware** | Junior Developer & Test Engineer at [Sunnyday Technologies](https://github.com/sunnyday-technologies)

## Play Now

Open `index.html` in any modern browser — no install, no downloads, no dependencies.

## What Is Egg Wars?

Egg Wars is a kid-friendly 3D voxel game inspired by popular Minecraft mini-game formats. Two teams each have an Egg on their floating island. Destroy the enemy's egg to win!

**The twist:** Instead of grinding for resources, you **earn them by solving challenges:**

- **Iron Station** (Easy) — Addition and subtraction problems
- **Gold Station** (Medium) — Multiplication and larger addition
- **Diamond Station** (Hard) — Multiplication and division

Use your resources to buy blocks, weapons, and armor at the Shop. Build a bridge across the void to the enemy island and destroy their egg!

## Game Modes

- **Math Challenge** — Earn resources by solving math problems
- **Spelling Challenge** — Earn resources by spelling words correctly
- **Mixed Mode** — Both math and spelling challenges

| Action | Key |
|--------|-----|
| Move | WASD |
| Look | Mouse |
| Jump | Space |
| Sprint | Shift |
| Attack / Mine | Left Click |
| Place Block | Right Click |
| Hotbar | 1-9 or Scroll |
| Shop | E (near gold block) |
| Challenge | E (near glowing station) |
| Use Item | Q |

## Tech

- Single HTML file — zero external dependencies
- WebGL 3D voxel engine built from scratch
- Procedural audio via Web Audio API
- Per-vertex ambient occlusion
- Procedural sky with stars
- All art is procedurally generated (no image assets)
